# ACRE Neobank — Mockup y levantamiento interactivo

Prototipo de referencia para la plataforma **ACRE Neobank**, elaborado por **ElipSoft LLC**.

## Contenido

- **`index.html`** — Levantamiento interactivo: recorrido por los 6 módulos de ACRE con preguntas de alcance por módulo, estados de avance y exportación de acta.
- **`pantallas.html`** — Mockup de pantallas del MVP: app del productor (Siembra Fácil) + backoffice administrativo.

## Ver en línea

Si GitHub Pages está activado, el levantamiento interactivo abre directamente en:

```
https://USUARIO.github.io/REPO/
```

(reemplaza `USUARIO` y `REPO` por los tuyos)

## Activar GitHub Pages

1. En el repositorio: **Settings → Pages**
2. En *Source*, selecciona la rama `main` y carpeta `/ (root)`
3. Guarda. La URL pública aparece en 1–2 minutos en esa misma pantalla.

## Notas

- Las respuestas del levantamiento viven en la sesión del navegador. Usa **"Exportar acta"** para guardarlas antes de cerrar.
- Diseño y contenido referencial para alineación con el cliente; no es el diseño final de producto.
